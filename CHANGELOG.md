All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-28

### Added
- Initial release
- `cool-branch add` - Adds both branch and worktree
- `cool-branch rm` - Removes both branch and worktree
- `cool-branch list` - List worktrees and branches
- `cool-branch dirname` - Configure directory in base directory used for repo worktrees
- `--base` - Support custom base directroy
- `cool-branch.sh` + `--setup` - Post worktree setup scripts
